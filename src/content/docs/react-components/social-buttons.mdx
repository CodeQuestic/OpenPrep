title: Social Buttons
---

The following component is for showcasing icons which redirect to social platforms. It is a single component and can be easily customized in terms of icons and also style the list itself and set the orientation. You can define more platforms in component file under ` platformStyles`, to use the button list you have to pass in an array with `platform`, `icon` and `url`. You can pass in any icon.
